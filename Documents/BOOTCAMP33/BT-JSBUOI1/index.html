Bài 1 : Tính Tiền Lương Nhân Viên

Input : lương 1 ngày và số ngày làm

Process :(pseudo code)
1. Lấy được input (hardcode)
2. Tính lương = lương 1 ngày * số ngày làm
3. log lương ra màn hình

OutPut : tính lương

Bài 2 : TÍNH GIÁ TRỊ TRUNG BÌNH

INPUT : nhập 5 số thực

PROGESS :
1. Lấy input
2. Tính tổng 5 số = cộng 5 số lại với nhau
3 . log ra trung bình 5 số = tổng / 5

OUTPUT: Tính giá trị trung bình

Bài 3 : QUY ĐỔI TIỀN

INPUT : nhập giá USD hiện tại

PROGESS :
1. Lấy input
2. Nhập số USD cần quy đổi sang VND
3 . log ra số tiền quy đổi = số USD * giá đô

OUTPUT: Tính Quy đổi tiền

Bài 4 : CHU VI VÀ DIỆN TÍCH HÌNH CHỮ NHẬT

INPUT : nhập chiều dài và chiều rộng

PROGESS :
1. Lấy input
2. Nhập chu vi = (dài + rộng) * 2
3. Nhập diện tích = dài * rộng
3 . log ra chu vi và diện tích

OUTPUT: Tính chu vi và diện tích

Bài 5 : TÍNH TỔNG 2 KÝ SỐ

INPUT : nhập 1 số có 2 chữ số bất kỳ

PROGESS :
1. Lấy input
2. Lấy hàng đơn vị = số trên input % 10
3. Lấy hàng đơn chục = số trên input / 10
3 . log ra tổng 2 ký số

OUTPUT: Tính tổng 2 ký số
